# Arabic Fonts for PDF Generation

To support Arabic text in PDF reports, you need to add an Arabic font file to this directory.

## Recommended Fonts

1. **Amiri** (Recommended)
   - Download from: https://fonts.google.com/specimen/Amiri
   - Save as: `Amiri-Regular.ttf`

2. **Cairo**
   - Download from: https://fonts.google.com/specimen/Cairo
   - Save as: `Cairo-Regular.ttf`

3. **Noto Sans Arabic**
   - Download from: https://fonts.google.com/noto/specimen/Noto+Sans+Arabic
   - Save as: `NotoSansArabic-Regular.ttf`

## Installation

1. Download the font file (TTF format)
2. Place it in the `server/fonts/` directory
3. The system will automatically detect and use it

## Alternative

If no Arabic font is found, the system will use the default font (Helvetica), but Arabic text may not render perfectly.
