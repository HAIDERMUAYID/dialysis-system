# دليل التثبيت والتشغيل

## المتطلبات الأساسية
- Node.js (الإصدار 14 أو أحدث)
- npm أو yarn

## خطوات التثبيت

### 1. تثبيت المتطلبات (Backend)
```bash
npm install
```

### 2. تثبيت المتطلبات (Frontend)
```bash
cd client
npm install
cd ..
```

### 3. تشغيل النظام

#### تشغيل Backend و Frontend معاً (موصى به):
```bash
npm run dev
```

#### أو تشغيل كلاهما بشكل منفصل:

في نافذة terminal الأولى:
```bash
npm run server
```

في نافذة terminal الثانية:
```bash
npm run client
```

### 4. الوصول إلى النظام
- الواجهة الأمامية: http://localhost:3000
- API الخلفية: http://localhost:5000

## الحسابات الافتراضية

### موظف الاستعلامات
- اسم المستخدم: `inquiry`
- كلمة المرور: `inquiry123`

### موظف التحليلات
- اسم المستخدم: `lab`
- كلمة المرور: `lab123`

### الصيدلي
- اسم المستخدم: `pharmacist`
- كلمة المرور: `pharmacist123`

### الطبيب
- اسم المستخدم: `doctor`
- كلمة المرور: `doctor123`

## بنية قاعدة البيانات

يتم إنشاء قاعدة البيانات تلقائياً عند التشغيل الأول. قاعدة البيانات هي SQLite وتوجد في ملف `database.sqlite` في المجلد الرئيسي.

## ملاحظات

- جميع البيانات محفوظة في قاعدة بيانات SQLite محلية
- عند أول تشغيل، سيتم إنشاء الحسابات الافتراضية تلقائياً
- قاعدة البيانات ستنشأ تلقائياً عند تشغيل الخادم لأول مرة
