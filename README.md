# نظام إدارة مستشفى الحكيم - شعبة الكلية الصناعية

نظام إدارة متكامل للمستشفى يحتوي على أربع مراحل رئيسية:

## المراحل الرئيسية

### المرحلة الأولى: موظف الاستعلامات
- إدارة المراجعين (إضافة/حذف)
- فتح زيارة جديدة للمراجع
- إنشاء ملف باسم المراجع

### المرحلة الثانية: موظف التحليلات
- استقبال الزيارات الجديدة
- إضافة نتائج التحاليل
- إغلاق الزيارة بعد إكمال التحاليل

### المرحلة الثالثة: الصيدلي
- استقبال الحالات بعد التحاليل
- تسجيل الأدوية المصروفة
- إغلاق الزيارة بعد صرف العلاج

### المرحلة الرابعة: الطبيب
- الاطلاع على جميع المعلومات
- عرض تاريخ الزيارات والتحاليل
- إعطاء التشخيص النهائي
- إنشاء التقارير الطبية

## التثبيت والتشغيل

### المتطلبات
- Node.js (الإصدار 14 أو أحدث)
- npm

### خطوات التثبيت

```bash
# تثبيت جميع المتطلبات (Backend + Frontend)
npm run install-all

# أو تثبيت كل واحد على حدة:
npm install                    # Backend
cd client && npm install       # Frontend
cd ..
```

### تشغيل النظام

```bash
# تشغيل النظام (الخلفية + الواجهة معاً)
npm run dev

# أو تشغيل كل واحد على حدة:
npm run server    # Backend على المنفذ 5000
npm run client    # Frontend على المنفذ 3000
```

## الوصول إلى النظام

- الواجهة الأمامية: http://localhost:3000
- API الخلفية: http://localhost:5000

## الحسابات الافتراضية

- **موظف الاستعلامات**: inquiry / inquiry123
- **موظف التحليلات**: lab / lab123
- **الصيدلي**: pharmacist / pharmacist123
- **الطبيب**: doctor / doctor123

## ميزات النظام

### موظف الاستعلامات
- إدارة المرضى (إضافة/تعديل/حذف)
- إنشاء زيارة جديدة للمريض
- عرض قائمة المرضى

### موظف التحليلات
- عرض الزيارات المعلقة
- إضافة نتائج التحاليل
- تعديل وحذف نتائج التحاليل
- إكمال التحاليل وإرسال الحالة للصيدلي

### الصيدلي
- عرض الزيارات بعد إكمال التحاليل
- إضافة الأدوية المصروفة
- تعديل وحذف الأدوية
- صرف العلاج وإرسال الحالة للطبيب

### الطبيب
- البحث عن المرضى بالاسم أو رقم الهوية
- عرض تاريخ الزيارات للمريض
- الاطلاع على جميع التفاصيل (التحاليل، الأدوية، التاريخ)
- إضافة التشخيص
- إكمال الزيارة
- إنشاء تقرير طبي PDF

## بنية المشروع

```
hosptal/
├── server/                 # Backend (Node.js + Express)
│   ├── routes/            # مسارات API
│   ├── database/          # إعدادات قاعدة البيانات
│   ├── middleware/        # Middleware للمصادقة
│   └── index.js           # نقطة بداية الخادم
├── client/                # Frontend (React + TypeScript)
│   ├── src/
│   │   ├── components/    # مكونات React
│   │   ├── context/       # React Context
│   │   └── App.tsx        # المكون الرئيسي
│   └── public/            # الملفات العامة
├── database.sqlite        # قاعدة البيانات (يتم إنشاؤها تلقائياً)
└── package.json           # إعدادات المشروع الرئيسية
```

## التقنيات المستخدمة

### Backend
- Node.js + Express
- SQLite (قاعدة بيانات)
- JWT (المصادقة)
- bcryptjs (تشفير كلمات المرور)
- PDFKit (إنشاء التقارير)

### Frontend
- React + TypeScript
- React Router
- Axios (للتواصل مع API)
- CSS (تصميم عربي RTL)
